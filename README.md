## 🔬 核心技术

### 感知系统
- **HC-SR04 超声波测距**: 基于 TOF 原理，测量范围 2–400 cm
- **DHT11 温度补偿**: v = 331.45 + 0.61·t
- **CSI 单目摄像头**: 320×240 @ ~30fps
- **视觉检测**: HSV 色彩空间 + 形态学处理 + 最大轮廓筛选

### 控制算法
- **增量式 PID**: 速度-转向双闭环
- **差速合成**: vL = speed + steer, vR = speed - steer
- **四线程并发**: 主状态机 / 视觉处理(~30fps) / 传感器采集(~20Hz) / PID控制(~20Hz)

## 📊 实验结果

### 温度补偿效果

| 指标 | 补偿前 (v=340m/s) | 补偿后 (v=347m/s) |
|------|-------------------|-------------------|
| **误差范围** | −2.4 ~ −0.5 cm (−3.2% ~ −2.6%) | −0.3 ~ +0.1 cm (−0.4% ~ +0.6%) |
| **平均绝对误差** | 1.20 cm (2.94%) | **0.14 cm (0.34%)** |

### 场景测试（5 项全部达标）

| 场景 | 方法 | 指标 | 结果 |
|------|------|------|:--:|
| 直线跟随 | 0.5 m/s 匀速 | 超调 <10 cm | ✅ |
| 弯道跟随 | R=2m 圆弧 | 丢失 ≤1 次/圈 | ✅ |
| 速度自适应 | 0.2–1.0 m/s | 响应 <0.5s | ✅ |
| 目标丢失恢复 | 离开视野 ~3s | 恢复率 ≥90% | ✅ (100%) |
| 近距离防撞 | <30 cm 触发 | 无碰撞 | ✅ |
